# TypeBlitz

A fast, minimal typing test platform inspired by Monkeytype. Built for speed, accuracy, and distraction-free practice.

[Live Demo](https://typeblitz-flax.vercel.app/)

---

## Overview

TypeBlitz delivers a focused typing experience with real-time performance metrics, a clean interface, and support for multiple typing modes. Designed for both casual practice and serious speed training.

![TypeBlitz Preview](./public/assets/typeblitz.png)

---

## Features

| Feature | Description |
|---|---|
| **Real-time Metrics** | Live WPM, accuracy, and typing rhythm feedback |
| **Minimal Interface** | Clean layout with focus mode to eliminate visual noise |
| **Theme Support** | Dark and light modes for comfortable practice sessions |
| **Responsive Design** | Optimized for desktop, tablet, and mobile screens |
| **Code Typing** | Dedicated mode for programming syntax practice |
| **Performance Analytics** | Detailed stats with progress tracking via charts |

---

## Tech Stack

### Core
- **Next.js 16** — React framework with App Router
- **React 19** — UI library
- **TypeScript** — Type-safe development

### Styling & UI
- **Tailwind CSS** — Utility-first styling
- **Shadcn UI** — Component primitives

### Libraries
- **Monaco Editor** — Code editing experience
- **Recharts** — Data visualization for analytics
- **Framer Motion** — UI animations
- **Canvas Confetti** — Celebration effects

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/soumyaranjan-p/typeblitz.git

# Navigate to project directory
cd typeblitz

# Install dependencies
npm install

# Start development server
npm run dev
